hsl() and hsla()

  • MDN reference
  • caniuse Support

In this example, we are declaring three CSS Custom Properties in the :root selector:

  1. --hue
  2. --saturation
  3. --lightness

The three CSS Custom Properties are then combined in a hsl() function in the order requried by its h, s, and l arguments. The var() function allows us to do this, creating a light yellow color ().

Then, within each of the classes that are used to construct the warning banner example, we redefine the --lightness CSS Custom property to darken the yellow color.

Warning

That thing you did also did other things.

Try changing --hue's value in the :root selector in this Pen to adjust the warning banner's overall color. For example, using a value of 90 will turn everything different shades of green!
