            Nikola Divic • 6 years ago

            Thanks for an awesome article Tom!
            BEM is awesome but I have a small remark. Nesting blocks within blocks is not good practice and makes code harder to reuse, you should stick to nesting only elements. So instead of:

            <form class="”header__search-from”">
              <input class="”header__search-from__input”" type="”input”">
              <button class="”header__search-from__button”" type="”button”">
              </form>


            Make the .search-form a separate block so you can use it anywhere outside of the header:

            <form class="”search-form”">
              <input class="”search-form__input”" type="”input”" <button="" class="”search-form__button”" type="”button”">
            </form>


            If you want to add specific search-form styling for the header use a modifier, for example .search-form--header.
            Cheers! :)
